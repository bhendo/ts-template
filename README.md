# TS-TEMPLATE

A simple TypeScript project to use as a template

## Tooling

- TypeScript
- Webpack
- Prettier
- ESLint

## Getting started

Install the dev dependencies

```bash
npm install
```

Lint the project

```bash
npm run lint
```

Start development server with hot reloading

```bash
npm run start:dev
```

Build the project

```bash
npm run build
```
